OpenHMIS

 

The OpenHMIS project is an open source, standards-based HMIS (Homeless Management Information System); see the OpenHMIS API documentation for an introduction.

The current test server for the API is: hmis.opentechstrategies.com:8080

Try an endpoint like http://hmis.opentechstrategies.com:8080/openhmis/api/v3/clients. You'll need to pass an authorization header to get actual results. For now, anyone who signs in and passes that header can make read-only requests. Please post to the list if you would like authorization to make create/update/delete calls to this server (of course, you can also always set up your own). See this guide to getting an id token from Google and our OAuth flow documentation for more detail about authentication and authorization with the OpenHMIS API.

Alternatively, use our demo client app at http://hmis.opentechstrategies.com:3000. Sign in with a Google email account and try searching for clients.

API server source code and source code for the demo client are also available.

For more information about HMIS, see PCNI and our Google group.
