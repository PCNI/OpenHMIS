OpenHMIS

 

 

  • API documentation

  •  

  • The OpenHMIS project is an open source, standards-based HMIS (Homeless Management Information System).

  • The current test server for this API is: hmis.opentechstrategies.com:8080. Try an endpoint like http://hmis.opentechstrategies.com:8080/openhmis/api/v3/clients. You'll need to pass an authorization header to get actual results -- see this guide to getting an id token from Google and our OAuth flow documentation for more detail about authentication and authorization with the OpenHMIS API.

  • API server source code and source code for an out-of-date demo client are also available.

  • For more information about HMIS, see PCNI and our Google group.
